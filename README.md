# 🌍 Life Expectancy Prediction using Machine Learning

A Machine Learning regression project that predicts life expectancy using healthcare, economic, and social indicators.

This project focuses on analyzing how different factors influence lifespan and uses advanced regression models to generate accurate predictions.

---

# 📌 Project Overview

Life expectancy is influenced by multiple real-world factors such as:

* Healthcare quality
* Economic growth
* Immunization coverage
* Lifestyle conditions
* Disease prevalence

The main objective of this project is to:
✅ Analyze important health indicators
✅ Understand factors affecting lifespan
✅ Build predictive regression models
✅ Generate accurate life expectancy predictions

---

# 🧠 Machine Learning Problem Type

This is a **Supervised Machine Learning Regression Problem** because the target variable is numerical.

### Target Variable:

```text id="lpf6d9"
Life Expectancy
```

---

# 📂 Dataset Features

The dataset contains country-level healthcare and socio-economic statistics.

### Important features include:

* Country
* Year
* Adult Mortality
* Infant Deaths
* Alcohol Consumption
* BMI
* GDP
* Population
* Schooling
* Immunization Coverage
* HIV/AIDS Cases
* Healthcare Expenditure

---

# 🔍 Exploratory Data Analysis

EDA revealed several important insights:

✅ Developed countries generally have higher life expectancy
✅ Schooling strongly impacts lifespan positively
✅ Adult mortality negatively affects life expectancy
✅ GDP and healthcare expenditure improve lifespan prediction
✅ HIV/AIDS cases reduce expected lifespan significantly

### Data preprocessing steps:

* Handling missing values
* Feature scaling
* Encoding categorical variables
* Removing unnecessary columns
* Train-test splitting

# ⚙️ Models Used

The following regression models were trained and evaluated:

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor
* Ridge Regression
* Lasso Regression
* XGBoost Regressor

### Evaluation Metrics

* MAE (Mean Absolute Error)
* RMSE (Root Mean Squared Error)
* R² Score

# 📊 Model Performance

The selected regression model achieved strong prediction performance and accurately estimated life expectancy values on unseen data.

The model successfully captured relationships between:

* Healthcare quality
* Economic conditions
* Education
* Mortality indicators
* Disease prevalence

and lifespan prediction.
